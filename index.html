CODE ARDUINO KIT

Home Security System

CODE ARDUINO KIT

September - December 2019

  • This project is a part of Computer Design Course (CS 362) of UIC from Fall 2019 Semester.
  • This was a group project of three members.
  • The main idea of the project was to design and create a prototype of home security system.
  • The prototype is able to detect threats which could take place in a house in your absence.
  • Potential threats which system could detect includes theft, invade, smoke, fire, gas andlights.
  • The system is protected by a password which is you need to set every time you enable the system.
  • The security system is coded in C++ on Arduino and other hardwares.

Hardware and Concepts used:

Arduino UNO (3) I2C Communication BreadBoard (3) Mini BreadBoard (1) Batteries (3) 16x2 LCD Display (3) LED Lights (4) Photoresistor (1) Potentiometer (3) Pin Buttons (2) Ultrasonic Sensor (1) 4x4 Keypad (1) Smoke Sensor (1) Buzzers (3) Connecting Wires (~40) Resistors (8)
