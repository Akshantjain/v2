CODE Home Security System
August - December 2019

  • This project ia a part of Computer Design Course (CS 362) of UIC from Fall 2019 Semester.
  • This was a group project of three members.
  • The main idea of the project was to design and create a prototype of home security system.
  • The system is able to detect many threats which could take place in a house in your absence.
  • The potential threats which system could detect include theft, invade, smoke, fire, gas and lights.
  • The system is protected by a password which is you need to set every time you enable the system.
  • The security system is coded in C++ on Arduino and hardwares.

Hardware and Concepts used:

3 Arduinos I2C Communication 3 BreadBoards 1 Mini BreadBoard 3 Batteries 3 LCD Display 4 LED Lights 1 Photoresistor 1 Potentiometer 2 Pin Buttons Ultrasonic Sensor 4x4 Numpad Smoke Sensor 3 Buzzers Connecting Wires 8 Resistors
